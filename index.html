About Me

I’m Nuno Miguens, a front-end web developer and designer based in Castelo Branco, Portugal.

My background training is in communication design and my interest in web development started back in 2018 when I decided that it would be a logic step in my area to learn the basics in HTML, CSS and Javascript.

Fast-forward to today, I’ve created responsive websites, e-commerce platforms, small and medium web apps. My main focus these days is building accessible, inclusive products and digital experiences, hoping to get hired by a company where i can develop my skills and gain experience in larger and scalable projects.

My latest learning has been focused in going fullstack, here are a few technologies I’ve been working with recently:

Javascript Typescript React Redux Next MongoDB
